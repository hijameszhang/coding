# 从开发到部署上线
Node.js项目从开发到部署上线, 需要做的事项, 包含但不限于以下各项:
* 技术基础
  * Node.js
  * koa
  * koa-router
  * koa-views + 模板引擎(nunjucks)
  * mysql 或者 mongoose
  * redis + koa-redis)
  * koa-session / koa-generic-session
  * docker容器(按需选择)
  * pm2 + nginx
  * linux基础操作(可自行google/baidu学习)
* 项目开发
  * 项目结构规划
  * 开发工具选择&配置(vscode, eslint, git)
  * coding
* 部署上线
  * 域名 + 云服务器选购(一般会提供公网IP)
  * 云服务器环境搭建
  * 域名解析
  * 上线 + 验证测试
